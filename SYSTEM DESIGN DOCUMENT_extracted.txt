📘 SYSTEM DESIGN DOCUMENT (SDD)
Project Name
AegisGrid — AI-Optimized Privacy-Preserving Decentralized Microgrid Protocol
1️⃣ System Overview
AegisGrid is a distributed, multi-layer system composed of:
Multi-Agent Simulation & Reinforcement Learning Engine (Python)
Automated Market Maker Smart Contracts (Solidity)
Zero-Knowledge Privacy Layer (Circom + SnarkJS)
WebSocket-Based Integration Layer (Node.js)
Institutional Command Center Frontend (React + Three.js)
The system operates as a real-time, self-healing microgrid simulation with autonomous financial settlement.
2️⃣ High-Level Architecture
React + Three.js Frontend          ↓ (WebSocket)Node.js API Gateway (Express + Ethers)          ↓-----------------------------------------| Python RL Engine (DDPG + Simulation) || Ethereum Smart Contracts (AMM)       |-----------------------------------------          ↓zk-SNARK Proof Verification

uint deltaY = (stableReserve * deltaX)               / (energyReserve + deltaX);
Apply swapFee
Update reserves
Emit SwapEvent
addLiquidity()

Update k
slash()

3.3 Oracle Simulation Design
Since hardware is not used:
Oracle = Python service.
Flow:
Node commits energy.
Python simulation verifies injection.
Node.js sends confirmation to contract.
If false → slash() called.
